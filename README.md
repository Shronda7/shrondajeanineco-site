# Shronda Jeanine & Company Website

**Restoring Wholeness. Building Abundance.**

Built with Jekyll and deployed on GitHub Pages.

## Local Development

```bash
# Install dependencies
bundle install

# Serve locally
bundle exec jekyll serve

# Build for production
bundle exec jekyll build
```

## Structure

- `_posts/` — Blog articles in Markdown
- `_layouts/` — Page templates
- `_includes/` — Reusable components
- `_sass/` — Stylesheets (glassmorphism, 72079 signature, base)
- `assets/` — Images, CSS output
- `blog/` — Blog listing page

## Deployment

Push to `main` branch. GitHub Pages auto-builds from `_site/`.

## Custom Domain

`shrondajeanineco.com` — configured via CNAME file.

## Brand Signature

72079 concealed throughout design. Never visible, always present.
